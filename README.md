# Embedded Protocol System

Система эмуляции двусторонней связи между Device A (Raspberry Pi) и Device B (контроллер) по виртуальному последовательному порту.

## Особенности

- Потокобезопасный виртуальный канал связи
- Бинарный протокол с CRC-16/X25
- Сериализация/десериализация пакетов
- Логирование в файл и консоль
- Корректная обработка ошибок
- Многопоточная архитектура
- Юнит-тесты
